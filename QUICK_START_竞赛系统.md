## 🎯 积分系统

### 得分公式

```
总得分 = (基础分 + 难度加成 + 速度奖励 + 连击奖励 + 首次奖励) × 连击倍数
```

### 示例

**场景**: 难度3单词,答题2秒,当前5连击,首次答对

```
基础分 = 10
难度加成 = 3 × 5 = 15
速度奖励 = 10 (2秒内完成)
连击奖励 = 5 × 2 = 10
首次奖励 = 5
连击倍数 = 2.0 (5连击)

总得分 = (10 + 15 + 10 + 10 + 5) × 2.0 = 100分 🎉
```

### 连击系统

| 连击数 | 倍数 | 显示 |
|-------|------|------|
| 1题 | ×1.0 | - |
| 2-4题 | ×1.5 | 🔥 |
| 5-9题 | ×2.0 | 🔥🔥 |
| 10-19题 | ×3.0 | 🔥🔥🔥 |
| 20+题 | ×3.0 | 🔥🔥🔥🔥🔥 |

**注意**: 答错一题,连击清零!

## 🔍 故障排查

### 问题: 进入竞赛页面后白屏

**原因**: Token获取失败

**解决**:
```javascript
// 浏览器控制台检查
localStorage.getItem('access_token')

// 如果为null,重新登录
```

### 问题: WebSocket连接失败

**现象**: 控制台显示 `WebSocket connection failed`

**解决**:
1. 确保后端已启动: http://localhost:8000/docs
2. 检查CORS配置是否正确
3. 确认使用uvicorn启动(不要用gunicorn)

### 问题: 排行榜不显示

**解决**:
```bash
# 检查数据库
sqlite3 english_helper.db "SELECT * FROM competition_seasons;"

# 应该看到一条记录:
# 1|2025年度总赛季|special|...
```

### 问题: 提交答题失败

**现象**: 点击选项后报错

**解决**:
1. 检查后端日志
2. 确认路由已注册 (访问 /docs 看是否有 "竞赛系统")
3. 查看浏览器Network标签,看具体错误

## 📊 测试数据

### 创建多个测试用户

```python
# backend/create_test_users.py
import asyncio
from sqlalchemy import select
from app.core.database import get_db
from app.models.user import User
from passlib.context import CryptContext

pwd_context = CryptContext(schemes=['bcrypt'], deprecated='auto')

async def create_users():
    async for db in get_db():
        # 创建5个学生用户
        for i in range(1, 6):
            # 检查是否已存在
            stmt = select(User).where(User.username == f'student{i}')
            result = await db.execute(stmt)
            if result.scalar_one_or_none():
                print(f'用户 student{i} 已存在')
                continue

            user = User(
                username=f'student{i}',
                password_hash=pwd_context.hash('123456'),
                nickname=f'学生{i}号',
                role='student',
                grade='小学3年级'
            )
            db.add(user)

        await db.commit()
        print('✅ 测试用户创建完成!')
        break

asyncio.run(create_users())
```

运行:
```bash

python create_test_users.py
```

### 生成测试答题数据

```python
# backend/generate_competition_data.py
import asyncio
import random
from app.core.database import get_db
from app.services.competition_service import competition_service

async def generate_data():
    async for db in get_db():
        # 为用户1-5生成随机答题记录
        for user_id in range(1, 6):
            score = 0
            for i in range(random.randint(10, 50)):
                is_correct = random.random() > 0.2  # 80%正确率
                result = await competition_service.submit_answer(
                    db=db,
                    user_id=user_id,
                    word_id=(i % 100) + 1,
                    is_correct=is_correct,
                    time_spent_ms=random.randint(1500, 5000),
                    question_type='choice',
                    season_id=1
                )
                score += result['result']['total_score']

            print(f'用户{user_id} 生成完成,总分: {score}')

        print('✅ 测试数据生成完成!')
        break

asyncio.run(generate_data())
```

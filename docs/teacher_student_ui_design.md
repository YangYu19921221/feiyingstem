# 教师端-学生端UI打通设计方案

**日期**: 2025-11-21
**目标**: 实现教师按单元录入单词,学生按单元学习的完整流程

---

## 🎯 核心设计理念

### 教师端工作流程
```
1. 创建单词本 → 2. 创建单元 → 3. 录入单词到单元 → 4. 分配给学生
```

### 学生端学习流程
```
1. 查看分配的单词本 → 2. 选择单元 → 3. 学习模式选择 → 4. 断点续学
```

## 🔄 三、数据流设计

### 3.1 教师端创建流程

```
步骤1: 创建单词本
POST /api/v1/words/books
{
  "name": "小学三年级上册",
  "grade_level": "Grade 3"
}
响应: book_id = 1

步骤2: 创建单元
POST /api/v1/teacher/books/1/units
{
  "unit_number": 1,
  "name": "Unit 1: Colors"
}
响应: unit_id = 1

步骤3: 添加单词到单元
POST /api/v1/teacher/units/1/words
{
  "word_ids": [1, 2, 3, 4, 5]
}
响应: success_count = 5

步骤4: (可选)分配给学生
POST /api/v1/teacher/assignments
{
  "book_id": 1,
  "student_ids": [101, 102, 103]
}
```

---

### 3.2 学生端学习流程

```
步骤1: 获取我的单词本
GET /api/v1/student/books
响应: [
  {
    "id": 1,
    "name": "小学三年级上册",
    "unit_count": 5,
    "word_count": 120,
    "progress_percentage": 35
  }
]

步骤2: 查看单元列表
GET /api/v1/student/books/1/units
响应: [
  {
    "id": 1,
    "name": "Unit 1: Colors",
    "word_count": 12,
    "completed_words": 9,
    "progress_percentage": 75,
    "has_progress": true,
    "current_word_index": 8
  }
]

步骤3: 开始/继续学习
POST /api/v1/student/units/1/start
{
  "learning_mode": "flashcard"
}
响应: {
  "has_existing_progress": true,
  "current_word_index": 8,
  "completed_words": 9,
  "total_words": 12,
  "words": [...12个单词数据...],
  "message": "继续上次的学习,从第9个单词开始"
}

步骤4: 学习过程中更新进度
PUT /api/v1/student/progress
{
  "unit_id": 1,
  "learning_mode": "flashcard",
  "current_word_index": 9,
  "current_word_id": 10,
  "word_result": "know" // know/dont_know
}

步骤5: 完成学习
PUT /api/v1/student/progress
{
  "unit_id": 1,
  "learning_mode": "flashcard",
  "current_word_index": 12,
  "is_completed": true
}
```

## 📊 五、关键数据表

### learning_progress (断点续学核心表)

```sql
user_id: 1                    -- 学生ID
unit_id: 1                    -- 单元ID
learning_mode: 'flashcard'    -- 学习模式
current_word_index: 8         -- 当前学到第8个单词 ⭐
current_word_id: 9            -- 当前单词的ID
completed_words: 9            -- 已完成9个单词
total_words: 12               -- 总共12个单词
is_completed: false           -- 未完成
last_studied_at: 2025-11-21   -- 最后学习时间
```

**断点续学算法**:
```python
# 获取进度
progress = get_learning_progress(user_id, unit_id, mode)

if progress is None:
    # 第一次学习
    current_index = 0
else:
    # 断点续学
    current_index = progress.current_word_index

# 显示提示
if current_index > 0:
    show_message(f"继续上次的学习,从第{current_index + 1}个单词开始")
```

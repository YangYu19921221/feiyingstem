# 🚀 实时竞赛系统快速启动指南

按照以下步骤,5分钟内启动完整的实时竞赛排名系统!

---

## 第一步: 安装依赖

### 后端依赖
```bash
cd backend
pip install websockets redis
```

### 前端依赖
```bash
cd frontend
npm install
```


## 第三步: 注册API路由

编辑 `backend/app/main.py`,在路由注册部分添加:

```python
from app.api.v1 import competition

# 在现有路由后添加
app.include_router(
    competition.router,
    prefix="/api/v1/competition",
    tags=["竞赛系统"]
)
```

**完整示例**:
```python
# backend/app/main.py
from fastapi import FastAPI
from app.api.v1 import auth, words, ai, learning, exams
from app.api.v1 import competition  # 新增

app = FastAPI(title="英语助手API")

# 注册路由
app.include_router(auth.router, prefix="/api/v1/auth", tags=["认证"])
app.include_router(words.router, prefix="/api/v1/words", tags=["单词"])
app.include_router(ai.router, prefix="/api/v1/ai", tags=["AI"])
app.include_router(learning.router, prefix="/api/v1/learning", tags=["学习"])
app.include_router(exams.router, prefix="/api/v1/exams", tags=["考试"])
app.include_router(competition.router, prefix="/api/v1/competition", tags=["竞赛系统"])  # 新增
```

## 第四步: 更新用户模型

编辑 `backend/app/models/user.py`,在 `User` 类中添加关系:

```python
from sqlalchemy.orm import relationship

class User(Base):
```

## 🔍 故障排查

### 问题1: WebSocket连接失败

**现象**: 前端控制台显示 `WebSocket connection failed`

**解决**:
```bash
# 检查后端是否启动
curl http://localhost:8000/api/v1/competition/online-users

# 检查端口是否被占用
lsof -i :8000

# 确保使用uvicorn启动(不要用gunicorn)
uvicorn app.main:app --reload
```

### 问题2: 提交答题失败

**现象**: 点击选项后报错 `401 Unauthorized`

**解决**:
```javascript
// 检查token是否存在
console.log(localStorage.getItem('access_token'));

// 重新登录获取新token
```

### 问题3: 排行榜不显示数据

**现象**: 排行榜为空或一直加载

**解决**:
```bash
# 检查数据库是否有数据
cd backend
sqlite3 english_helper.db "SELECT * FROM user_scores;"

# 手动插入测试数据
sqlite3 english_helper.db "
INSERT INTO competition_seasons (name, season_type, start_time, end_time, is_active)
VALUES ('测试赛季', 'special', '2025-01-01', '2025-12-31', 1);
"
```

### 问题4: 前端组件找不到

**现象**: `Module not found: Can't resolve './components/LiveLeaderboard'`

**解决**:
```bash
# 检查文件是否存在
ls frontend/src/components/

# 应该看到:
# LiveLeaderboard.tsx
# AnswerFeedback.tsx
# RankNotification.tsx

# 检查路径是否正确
cd frontend/src
tree components/
```

### 问题5: 实时通知不显示

**现象**: 答题后没有右上角通知

**解决**:
```javascript
// 检查WebSocket事件监听
competitionWS.on('*', (msg) => {
  console.log('收到消息:', msg);
});

// 确保RankNotification组件已添加到页面
```

## 📊 测试数据生成

快速生成测试数据:

```python
# backend/generate_test_data.py
import asyncio
from app.core.database import get_db
from app.services.competition_service import competition_service

async def generate_test_data():
    async for db in get_db():
        # 为用户1生成随机答题记录
        for i in range(20):
            await competition_service.submit_answer(
                db=db,
                user_id=1,
                word_id=i + 1,
                is_correct=True if i % 3 != 0 else False,
                time_spent_ms=2000 + i * 100,
                question_type='choice',
                season_id=1
            )

        print('✅ 测试数据生成成功!')
        break

asyncio.run(generate_test_data())
```

运行:
```bash
cd backend
python generate_test_data.py
```

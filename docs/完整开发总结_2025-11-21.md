   ```python
   - StudySessionCreate        # 创建学习会话
   - StudySessionUpdate        # 更新学习会话
   - WordMasteryResponse       # 单词掌握度响应
   ```

2. **API端点** (`backend/app/api/v1/student/learning_records.py`)

**新增API端点**:

**学习记录API**:
- `POST /api/v1/student/records` - 批量创建学习记录
  - 自动更新单词掌握度
  - 自动更新学习日历
  - 支持批量提交

**学习会话API**:
- `POST /api/v1/student/sessions` - 创建学习会话
- `PUT /api/v1/student/sessions/{id}` - 更新学习会话

**单词掌握度API**:
- `GET /api/v1/student/mastery/{word_id}` - 查询单个单词掌握度
- `GET /api/v1/student/mastery` - 查询所有单词掌握度
- `GET /api/v1/student/weak-words` - 查询薄弱单词(掌握度<3)
- `GET /api/v1/student/review-due` - 查询需要复习的单词

**核心算法**:

1. **掌握度等级算法 (0-5级)**:
   ```
   Level 5: 正确率 ≥95% 且遇到 ≥5次  (完全掌握)
   Level 4: 正确率 ≥85% 且遇到 ≥4次  (基本掌握)
   Level 3: 正确率 ≥75% 且遇到 ≥3次  (一般掌握)
   Level 2: 正确率 ≥60% 且遇到 ≥2次  (初步掌握)
   Level 1: 遇到 ≥1次                (刚接触)
   Level 0: 未学习
   ```

2. **间隔重复算法**:
   ```
   Level 4-5: 7天后复习  (掌握良好,延长间隔)
   Level 2-3: 3天后复习  (一般掌握,中等间隔)
   Level 0-1: 1天后复习  (掌握较差,短期复习)
   ```

3. **多模式统计**:
   - 分别记录 flashcard、quiz、spelling、fillblank 四种模式
   - 每种模式独立统计正确/错误次数
   - 支持针对性薄弱模式分析

## 技术亮点

### 1. 智能掌握度系统
- **动态评估**: 根据正确率和练习次数双重指标评估
- **等级划分**: 0-5级精细化掌握度划分
- **实时更新**: 每次答题自动更新掌握度

### 2. 间隔重复学习
- **科学复习**: 基于艾宾浩斯遗忘曲线
- **个性化间隔**: 根据掌握度自动调整复习时间
- **智能提醒**: API支持查询需要复习的单词

### 3. 多维度数据分析
- **班级视角**: 整体趋势、平均水平、活跃度
- **学生视角**: 个人进度、薄弱点、学习习惯
- **单词视角**: 难度分析、易错统计
- **模式视角**: 不同学习模式效果对比

### 4. 批量操作优化
- **批量记录**: 一次请求提交多个单词的学习记录
- **性能优化**: 减少API调用次数,提高响应速度
- **事务保证**: 批量操作原子性,确保数据一致性

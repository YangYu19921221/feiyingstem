# AI试卷生成功能使用说明

## 功能概述

本系统提供了AI智能试卷生成功能,可根据学生的学习数据和薄弱点自动生成个性化试卷。

## 使用步骤

### 1. 进入学生详情页
- 教师端 → 数据分析 → 学生列表
- 点击任意学生的"查看详情"按钮

### 2. 查看学生学习数据
页面会显示以下数据卡片:
- 📚 **学习时长**: 总学习时间
- 📖 **已学单词**: 学习过的单词总数
- 🎯 **平均准确率**: 答题正确率
- ⚠️ **薄弱单词**: 掌握度较低的单词数量

### 3. 查看薄弱点详情
下方会列出学生的薄弱单词列表,包括:
- 单词和释义
- 错误次数和总尝试次数
- 准确率
- 出错的学习模式(卡片/选择题/拼写/填空)
- 最后一次错误时间

### 4. AI深度分析(可选)
点击"🔮 AI深度分析"按钮,系统会:
- 分析学习单词数和正确率
- 推荐试题难度(简单/中等/困难)
- 推荐题型分布(选择题/填空题/拼写题/阅读题)
- 列出Top 10薄弱单词

### 5. 一键生成试卷
点击"📄 一键生成试卷"按钮后:

#### 进度显示
系统会显示精美的进度弹窗,实时展示生成进度:

```
🔍 正在分析学生薄弱点... (15%)
📊 评估学习数据... (30%)
🎯 确定试题难度... (45%)
🤖 AI生成选择题... (60%)
✍️ AI生成填空和拼写题... (75%)
📖 AI生成阅读理解题... (90%)
✅ 整理试卷... (95%)
🎉 试卷生成完成! (100%)
```

#### 进度弹窗特性
- **渐变进度条**: 橙色→粉色→紫色渐变
- **流光效果**: 进度条带有动态光效
- **阶段提示**: 实时显示当前生成阶段
- **百分比显示**: 精确显示生成进度
- **背景模糊**: 半透明背景突出进度窗口
- **动画效果**: 平滑的淡入淡出和缩放动画

### 6. 预览和使用试卷
生成完成后自动跳转到试卷预览页面,可以:
- 查看所有题目
- 下载PDF试卷
- 打印试卷
- 分配给学生

## 题型说明

### 支持的题型
1. **选择题 (choice)**: 四选一单选题
2. **填空题 (fill_blank)**: 根据中文填写英文单词
3. **拼写题 (spelling)**: 听/看中文拼写英文
4. **翻译题 (translation)**: 英译中或中译英
5. **阅读理解 (reading)**: 短文阅读+问题

### 默认题型分布
- 总题数: 20题
- 选择题: 8题 (40%)
- 填空题: 6题 (30%)
- 拼写题: 4题 (20%)
- 阅读理解: 2题 (10%)

系统会根据学生在不同学习模式下的表现动态调整比例。

## AI生成策略

### 难度评估
| 准确率 | 推荐难度 | 说明 |
|--------|---------|------|
| < 60% | 简单 (easy) | 巩固基础,建立信心 |
| 60-80% | 中等 (medium) | 常规练习,稳步提升 |
| > 80% | 困难 (hard) | 挑战自我,拓展能力 |

### 单词选择
- 优先选择**掌握度最低**的单词
- 选择**最近练习过但出错**的单词
- 覆盖学生的**主要薄弱点**
- 最多选择15个核心薄弱单词

### 题型动态调整
```python
if 拼写错误率 > 50%:
    增加拼写题比例到 30%

if 选择题错误率 > 60%:
    增加选择题比例到 50%

if 填空题错误率 > 50%:
    增加填空题比例到 35%
```

## 常见问题

### Q: 生成失败怎么办?
A: 系统会显示错误提示,请检查:
- AI服务是否配置正确 (`.env`中的API Key)
- 学生是否有学习数据
- 网络连接是否正常

### Q: 进度条卡住不动?
A: 这可能是AI服务响应慢,请耐心等待。如果超过30秒,可以刷新页面重试。

### Q: 生成的题目质量如何?
A: AI会根据:
- 学生的实际薄弱点
- 单词的难度级别
- 适合的年级范围

来生成题目,确保针对性和适用性。

### Q: 可以调整题目数量吗?
A: 目前固定为20题,未来版本会支持自定义。

### Q: 试卷可以重复生成吗?
A: 可以,每次生成的题目都会有所不同,因为AI会实时分析最新的学习数据。

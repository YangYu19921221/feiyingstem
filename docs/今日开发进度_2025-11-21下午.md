### 2. 后端AI接口改造(核心功能)
- ✅ **新增基于单元的题目生成API** (`/api/v1/ai/generate-unit-quiz`)

  **功能特点**:
  - 题目只包含当前单元的单词(不再乱出题)
  - 支持3种题型:
    - `choice`: 选择题(看中文选英文)
    - `spelling`: 拼写题(听音写词)
    - `fillblank`: 填空题(根据例句填空)
  - 干扰选项优先从同单元其他单词中选取
  - 如果单元单词不足,调用AI生成补充

  **API请求示例**:
  ```json
  POST /api/v1/ai/generate-unit-quiz
  {
    "unit_id": 1,
    "question_count": 10,
    "question_type": "spelling"
  }
  ```

  **响应示例**:
  ```json
  {
    "unit_id": 1,
    "unit_name": "Unit 1: Colors",
    "questions": [
      {
        "word_id": 1,
        "word": "apple",
        "question": "请拼写单词: 苹果 [/ˈæpl/]",
        "options": null,
        "correct_answer": "apple",
        "explanation": "正确拼写: apple"
      },
      ...
    ],
    "total_count": 10
  }
  ```

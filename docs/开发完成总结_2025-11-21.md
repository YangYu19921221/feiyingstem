**文件**: `/backend/app/api/v1/ai.py`

**新增API**: `POST /api/v1/ai/generate-unit-quiz`

**功能**:
- 基于单元生成练习题(不再乱出题!)
- 支持3种题型: `choice`, `spelling`, `fillblank`
- 智能干扰项生成(优先同单元单词)

**API示例**:
```bash
# 生成选择题
curl -X POST http://localhost:8000/api/v1/ai/generate-unit-quiz \
  -H "Content-Type: application/json" \
  -d '{"unit_id":1, "question_count":10, "question_type":"choice"}'
```

## 🔗 系统架构

### 数据流
```
UnitSelector (单元选择)
    ↓ 点击学习模式按钮
    ↓
FlashCardLearning   ← mode=flashcard
SpellingPractice    ← mode=spelling
FillBlankPractice   ← mode=fillblank
QuizPractice        ← mode=quiz
    ↓ 调用AI接口
    ↓
POST /api/v1/ai/generate-unit-quiz
    ↓ 返回题目列表
    ↓
学生答题 → 统计得分 → 完成弹窗
```

### 路由结构
```
/student/dashboard
  → /student/books/:bookId/units
    → /student/units/:unitId/flashcard  (卡片)
    → /student/units/:unitId/spelling   (拼写)
    → /student/units/:unitId/fillblank  (填空)
    → /student/units/:unitId/quiz       (测试)
```

### 1. 后端数据分析API (300行)

**文件位置**: `/backend/app/api/v1/analytics.py`

**核心接口**:

#### A. GET `/api/v1/analytics/overview` - 学习总览
返回学生的整体学习数据:
```python
{
  "total_words": 156,           # 总学习单词数
  "mastered_words": 89,          # 已掌握(掌握度>=4)
  "learning_words": 45,          # 学习中(掌握度2-3)
  "weak_words": 22,              # 薄弱(掌握度<2)
  "total_study_days": 15,        # 总学习天数
  "total_duration": 5400,        # 总学习时长(秒)
  "avg_daily_words": 10.4,       # 日均学习单词数
  "current_streak": 7            # 当前连续打卡天数
}
```

**计算逻辑**:
- 掌握度分级: >=4已掌握, 2-3学习中, <2薄弱
- 连续打卡: 从今天往前倒推,检查每天是否有学习记录
- 日均单词: 总单词数 / 总学习天数

#### B. GET `/api/v1/analytics/daily-stats?days=30` - 每日统计
返回最近N天的学习数据,包括没有学习的日期:
```python
[
  {
    "date": "2025-11-21",
    "words_learned": 15,
    "duration": 1200,
    "accuracy": 0.0
  },
  {
    "date": "2025-11-20",
    "words_learned": 0,    # 这天没学习
    "duration": 0,
    "accuracy": 0.0
  },
  ...
]
```

**填充逻辑**:
```python
# 填充所有日期(包括没有学习的日期)
current_date = start_date
records_dict = {r.study_date: r for r in records}

while current_date <= date.today():
    record = records_dict.get(current_date)
    daily_stats.append(DailyStats(
        date=current_date.isoformat(),
        words_learned=record.words_learned if record else 0,
        duration=record.duration if record else 0,
        accuracy=0.0
    ))
    current_date += timedelta(days=1)
```

使用纯CSS实现的柱状图,无需图表库:

**特性**:

- ✅ 悬停显示详细数据(气泡提示)
- ✅ 渐进动画(每根柱子依次出现)
- ✅ 当日高亮(蓝紫渐变)
- ✅ Y轴刻度标签

**实现代码**:
```tsx
<div className="flex items-end justify-between h-full gap-1">
  {dailyStats.slice(-selectedDays).map((stat, index) => {
    const height = (stat.words_learned / maxDailyWords) * 100;
    const isToday = date.toDateString() === new Date().toDateString();

    return (
      <div className="flex-1 flex flex-col items-center justify-end group">
        {/* 提示气泡 */}
        <div className="absolute bottom-full opacity-0 group-hover:opacity-100 transition">
          <div className="bg-gray-800 text-white text-xs rounded-lg px-3 py-2">
            {stat.date}<br />
            {stat.words_learned} 个单词<br />
            {Math.floor(stat.duration / 60)} 分钟
          </div>
        </div>

        {/* 柱子 */}
        <motion.div
          initial={{ height: 0 }}
          animate={{ height: `${height}%` }}
          transition={{ delay: 0.5 + index * 0.02, duration: 0.3 }}
          className={`w-full rounded-t-md ${
            isToday
              ? 'bg-gradient-to-t from-blue-500 to-purple-500'
              : 'bg-gradient-to-t from-blue-400 to-cyan-400'
          }`}
        />

        {/* 日期标签 */}
        <p className="text-xs text-gray-500 mt-2">{date.getDate()}</p>
      </div>
    );
  })}
</div>
```

## 🔧 技术实现

### 1. 纯CSS图表
没有使用任何图表库(如Chart.js, Recharts),全部用CSS实现:

**优势**:
- ✅ 体积小,无额外依赖
- ✅ 性能好,GPU加速
- ✅ 完全可控,易于定制
- ✅ Framer Motion动画集成

**劣势**:
- ⚠️ 功能有限(无复杂交互)
- ⚠️ 需要手动计算比例

### 2. SVG圆环图
使用SVG的`<circle>`标签 + `stroke-dasharray`实现进度:

```typescript
const circumference = 2 * Math.PI * radius;
const progress = (value / total);
const dashoffset = circumference * (1 - progress);

<circle
  r={radius}
  strokeDasharray={circumference}
  strokeDashoffset={dashoffset}
/>
```

### 3. 响应式设计
```tsx
<div className="grid grid-cols-2 md:grid-cols-4 gap-4">  // 移动端2列,桌面4列
<div className="grid grid-cols-1 lg:grid-cols-2 gap-8">  // 移动端1列,大屏2列
```

### 4. 条件渲染
```tsx
{overview && overview.total_words > 0 && (
  <div>圆环图</div>
)}

{recentActivities.length === 0 ? (
  <p>暂无学习记录</p>
) : (
  <div>活动列表</div>
)}
```

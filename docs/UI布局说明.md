## 🎯 核心UI组件

### 1. 实时排行榜 (右侧固定)

**位置**: 页面右侧,宽度约400px,sticky定位

**功能**:
- ✅ 今日榜/本周榜/总榜切换
- ✅ 实时排名动画(平滑过渡)
- ✅ 高亮显示当前用户
- ✅ 排名徽章(👑🥈🥉🏅⭐)
- ✅ 分数进度条
- ✅ 在线人数/更新时间

### 2. 答题区域 (左侧主要区域)

**功能**:
- ✅ 单词展示(大号字体)
- ✅ 音标显示
- ✅ 难度等级
- ✅ 选项按钮(2×2网格)
- ✅ 悬停/点击动画

### 3. 个人统计卡片 (顶部)

**显示内容**:
- 今日排名
- 今日积分
- 最高连击
- 答题数
- 正确率

### 4. 答题反馈弹窗 (答题后弹出)

**内容**:
- ✅ 正确/错误图标
- ✅ 得分明细
- ✅ 排名变化动画
- ✅ 连击状态
- ✅ 关闭按钮

### 5. 实时通知 (右上角)

**位置**: 固定在屏幕右上角(top: 5rem, right: 1rem)

**类型**:
- 🎉 超越通知(绿色)
- ⚠️ 被超越通知(橙色)
- 🔥 连击里程碑(红色)

**特效**:
- 滑入动画
- 3秒后滑出
- 堆叠显示多个通知

## 🎮 交互流程

### 1. 进入页面
```
用户访问 /student/competition
    ↓
加载个人统计
    ↓
连接WebSocket
    ↓
接收排行榜数据
    ↓
加载第一题
```

### 2. 答题流程
```
用户点击选项
    ↓
记录答题时间
    ↓
POST /api/v1/competition/submit-answer
    ↓
显示答题反馈弹窗
    ↓
WebSocket推送排名更新 (所有在线用户)
    ↓
更新排行榜UI
    ↓
如果超越别人 → 发送通知给被超越者
    ↓
3秒后关闭反馈弹窗
    ↓
加载下一题
```

### 3. 实时更新
```
用户A答题 → 提交积分 → 排名上升
    ↓
WebSocket广播 rank_update
    ↓
所有在线用户收到更新
    ↓
排行榜平滑动画更新
    ↓
被超越的用户B收到 overtaken 通知
    ↓
右上角显示通知 "你被XXX超越了!"
```

## 🎨 颜色方案

### 主题色
- **主色**: `#FF6B35` (橙色)
- **辅色**: `#FFD23F` (黄色)
- **强调**: `#00D9FF` (蓝色)

### 排名颜色
- **第1名**: 金色渐变 `from-yellow-400 to-yellow-600`
- **第2名**: 银色渐变 `from-gray-300 to-gray-500`
- **第3名**: 铜色渐变 `from-orange-400 to-orange-600`
- **4-10名**: 蓝色渐变 `from-blue-400 to-blue-600`

### 通知颜色
- **超越**: 绿色 `from-green-500 to-emerald-600`
- **被超越**: 橙红色 `from-orange-500 to-red-600`

## 🚀 如何启用竞赛模式

### 方法1: 直接访问URL
```
http://localhost:5173/student/competition
```

### 方法2: 从学生仪表盘添加入口

编辑 `frontend/src/pages/StudentDashboard.tsx`,添加按钮:

```tsx
import { useNavigate } from 'react-router-dom';

function StudentDashboard() {
  const navigate = useNavigate();

  return (
    <div>
      {/* 现有内容 */}

      {/* 添加竞赛模式入口 */}
      <button
        onClick={() => navigate('/student/competition')}
        className="bg-gradient-to-r from-orange-500 to-red-500 text-white font-bold py-4 px-6 rounded-xl shadow-lg hover:shadow-xl transition-shadow"
      >
        🏆 进入竞赛模式
      </button>
    </div>
  );
}
```

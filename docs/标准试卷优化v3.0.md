# 标准试卷优化 v3.0

## 📝 更新时间
2025-11-23

## 🎯 优化目标
将试卷格式从简化版(20题)升级为符合标准中小学英语考试的专业格式(50题)

## 📊 优化对比

### 之前 v2.0 (❌ 不够专业)
```
总题数: 20题
├── 选择题: 8题 (40%)
├── 填空题: 6题 (30%)
├── 拼写题: 4题 (20%)
└── 阅读理解: 2题 (10%) ← 问题:太简陋
    └── 短文: 3-5句话
    └── 问题: 每篇1题
```

### 现在 v3.0 (✅ 符合标准)
```
总题数: 50题
├── 选择题: 15题 x 1分 = 15分
├── 完形填空: 10题 x 1分 = 10分 ⭐ 新增
│   └── 短文: 100-150词
│   └── 10个空,每空4个选项
├── 填空题: 8题 x 2分 = 16分
├── 拼写题: 5题 x 2分 = 10分
└── 阅读理解: 12题 x 3分 = 36分 ⭐ 改进
    ├── 文章1: 100-200词 + 4题
    ├── 文章2: 100-200词 + 4题
    └── 文章3: 100-200词 + 4题
───────────────────────────────────
总分: 87分 (可调整为100分)
```

## ✨ 主要改进

### 1. 新增完形填空题型 📄

**数据库层面:**
```sql
-- exam_questions表约束更新
question_type CHECK(question_type IN (
  'choice',
  'cloze_test',  -- 新增
  'fill_blank',
  'spelling',
  'reading'
))
```

**AI生成逻辑:**
- 生成100-150词的完整短文
- 挖空10个位置,使用 ___1___, ___2___ 等标记
- 每个空提供4个选项
- 考查上下文理解和词汇运用

**前端显示:**
- 一次性显示完整短文
- 短文中标记空白位置
- 分别列出10个小题及选项
- 支持答案切换显示

### 2. 大幅改进阅读理解 📖

**之前的问题:**
- 短文只有3-5句话(约30-50词)
- 每篇只有1个问题
- 总共2题,过于简单

**现在的改进:**
- **文章长度**: 100-200词的完整文章
- **文章数量**: 3篇独立文章
- **问题数量**: 每篇4题,共12题
- **问题类型**:
  - 细节理解
  - 推理判断
  - 主旨大意
  - 词义猜测

**实现方式:**
- 使用 `passage_id` 对问题分组
- 每组共享相同的文章
- 前端按文章分组显示

### 3. 调整题型分布

**数据结构:**
```typescript
export interface ExamQuestion {
  question_number: number;
  question_type: string;
  content: string;
  correct_answer: string;
  explanation?: string;
  score: number;

  // 阅读理解专用
  passage?: string;
  passage_id?: string;      // 用于分组
  passage_title?: string;   // 文章标题

  // 完形填空专用
  blanks?: Array<{
    blank_number: number;
    content: string;
    options: Array<{key: string; text: string}>;
    correct_answer: string;
    explanation: string;
    score: number;
  }>;
}
```

## 🔧 技术实现

**ai_service.py - 默认题型分布:**
```python
distribution = {

    "fill_blank": 8,     # 填空题
    "spelling": 5,       # 拼写题
    "reading": 12        # 阅读理解(扩展)
}
```

**AI提示词优化:**
```python
# 完形填空提示词
- 提供一篇100-150词的完整短文
- 短文中挖空10个位置,编号为 ___1___, ___2___ 等
- 每个空给出4个选项
- 短文要有完整的情节或主题

# 阅读理解提示词
- 生成3篇独立的短文,每篇100-200词
- 每篇短文要有完整的主题和情节
- 每篇短文配4个问题
- 问题类型包括:细节理解、推理判断、主旨大意、词义猜测
```

### 前端优化

**题目渲染特殊处理:**

1. **完形填空渲染:**
```typescript
if (type === 'cloze_test' && questions[0].blanks) {
  return (
    <>
      {/* 显示完整短文 */}
      <div className="passage">{question.passage}</div>

      {/* 列出10个空的题目 */}
      {question.blanks.map((blank, index) => (
        <div key={index}>
          <div>空 {blank.blank_number}</div>
          <div className="options">
            {blank.options.map(opt => (
              <div>{opt.key}. {opt.text}</div>
            ))}
          </div>
        </div>
      ))}
    </>
  )
}
```

2. **阅读理解渲染:**
```typescript
if (type === 'reading') {
  // 按passage_id分组
  const passageGroups = groupByPassageId(questions);

  return Object.entries(passageGroups).map(([passageId, passageQuestions]) => (
    <>
      {/* 显示文章标题和内容 */}
      <div className="passage-title">{passageQuestions[0].passage_title}</div>
      <div className="passage">{passageQuestions[0].passage}</div>

      {/* 显示该文章的所有问题 */}
      {passageQuestions.map(question => (
        <div>
          <div>{question.content}</div>
          <div className="options">...</div>
        </div>
      ))}
    </>
  ))
}
```

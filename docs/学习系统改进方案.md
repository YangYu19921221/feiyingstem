# 英语学习系统 - 学习流程改进方案

## 📋 当前问题分析

### 1. **学习模式缺乏关联性**
- ❌ 问题: AI测试、拼写、填空题目和当前单元单词无关联
- ❌ 结果: 学生可能遇到未学过的单词,降低学习效果
- ✅ 期望: 所有学习模式应基于当前单元的单词

### 2. **学习流程不够系统化**
- ❌ 问题: 各个学习模式相互独立,没有形成学习闭环
- ✅ 期望: 建立"学习→练习→测试→复习"的完整流程

---

## 🎯 改进方案 A: 渐进式学习流程(推荐)

### 设计理念
**"先认识 → 再练习 → 后测试"** - 符合学习规律的递进式设计

### 流程图
```
单元学习入口
    ↓
[第一步] 🃏 卡片认词 (必须完成)
    ↓ 完成后解锁
[第二步] 练习模式 (3选1或全做)
    ├─ ✏️ 拼写练习 (AI生成基于本单元单词)
    ├─ 📝 填空练习 (AI生成基于本单元单词)
    └─ 🎮 配对游戏 (新增:单词与释义配对)
    ↓ 完成任意一个解锁
[第三步] ✅ AI综合测试 (基于本单元单词)
    ↓
[完成] 🏆 获得成就徽章
```

### 详细设计

#### 阶段1: 卡片认词 (当前已实现)
- **目标**: 初次认识单词,建立基础记忆
- **模式**: 3D翻转卡片
- **机制**:
  - 正面显示单词+音标+发音
  - 背面显示释义+例句
  - 学生标记"认识/不认识"
  - 记录薄弱词汇(标记为"不认识"的词)
- **完成条件**: 浏览完所有单词

#### 阶段2: 强化练习 (需开发)

**A. 拼写练习 (✏️ AI辅助)**

## 🎯 改进方案 B: 自由选择模式(备选)

### 设计理念
保持当前设计,但**所有题目必须基于当前单元单词**

### 实现方式

#### 1. 后端API改造

**当前问题**: AI接口生成随机题目,与单元无关

**改进方案**:

```python
# app/api/v1/ai.py (修改)

@router.post("/generate-quiz", response_model=QuizResponse)
async def generate_quiz(
    request: QuizRequest,  # 新增unit_id参数
    db: AsyncSession = Depends(get_db)
):
    """
    生成选择题 - 基于指定单元的单词
    """
    # 1. 获取单元的所有单词
    words = await get_unit_words(db, request.unit_id)

    if len(words) < 4:
        raise HTTPException(400, "单元单词数不足,至少需要4个单词")

    # 2. 随机选择一个单词作为题目
    target_word = random.choice(words)

    # 3. 从同单元选择3个干扰项
    distractors = random.sample(
        [w for w in words if w.id != target_word.id],
        k=min(3, len(words)-1)
    )

    # 4. 如果干扰项不足3个,调用AI生成
    if len(distractors) < 3:
        ai_distractors = await ai_service.generate_distractors(
            word=target_word.word,
            meaning=target_word.meaning,
            count=3 - len(distractors),
            grade_level=request.grade_level
        )
        distractors.extend(ai_distractors)

    # 5. 组装题目
    return QuizResponse(
        question=f"'{target_word.word}' 的意思是?",
        options=shuffle([target_word.meaning] + [d.meaning for d in distractors]),
        correct_answer=target_word.meaning,
        word_id=target_word.id
    )
```

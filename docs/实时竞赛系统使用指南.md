# 实时竞赛排名系统使用指南

## 🎯 系统概述

基于**WebSocket**的实时竞赛排名系统,学生在答题背单词时可以看到自己的排名实时变化,并收到超越/被超越的即时通知。

### 核心特性

✅ **真正实时** - WebSocket推送,延迟<100ms
✅ **多维度排行榜** - 今日榜/周榜/总榜
✅ **游戏化激励** - 连击系统、排名徽章、超越通知
✅ **防作弊机制** - 答题时间验证、频率限制、异常检测
✅ **精准积分计算** - 基础分+难度+速度+连击+首次奖励

## 📡 API接口文档

### 1. WebSocket连接

**连接地址**: `ws://localhost:8000/api/v1/competition/ws/competition`

**查询参数**:
- `token` (必需): JWT认证token
- `season_id` (可选): 赛季ID,默认1

**示例**:
```javascript
const token = localStorage.getItem('token');
const ws = new WebSocket(
  `ws://localhost:8000/api/v1/competition/ws/competition?token=${token}&season_id=1`
);

ws.onopen = () => {
  console.log('连接成功');
};

ws.onmessage = (event) => {
  const message = JSON.parse(event.data);
  console.log('收到消息:', message);
};
```

**消息类型**:

| 类型 | 方向 | 说明 | 数据结构 |
|------|------|------|----------|
| `connected` | 服务端→客户端 | 连接成功 | `{message, leaderboard}` |
| `rank_update` | 服务端→客户端 | 排名变化 | `{user_id, nickname, old_rank, new_rank, score_delta}` |
| `leaderboard_update` | 服务端→客户端 | 排行榜更新 | `{rankings, total, online_users}` |
| `overtake` | 服务端→客户端 | 超越通知 | `{overtaken_name, new_rank, message}` |
| `overtaken` | 服务端→客户端 | 被超越通知 | `{overtaker_name, new_rank, message}` |
| `combo_milestone` | 服务端→客户端 | 连击里程碑 | `{combo, multiplier, message}` |
| `ping` | 服务端→客户端 | 心跳检测 | `{}` |
| `pong` | 客户端→服务端 | 心跳响应 | `{}` |
| `get_leaderboard` | 客户端→服务端 | 请求排行榜 | `{board_type: 'daily'/'weekly'/'overall'}` |

### 2. 答题提交

```tsx
import { useState } from 'react';
import axios from 'axios';
import AnswerFeedback from './components/AnswerFeedback';

function QuestionCard() {
  const [showFeedback, setShowFeedback] = useState(false);
  const [feedbackData, setFeedbackData] = useState(null);

  const handleAnswer = async (wordId: number, isCorrect: boolean, timeSpent: number) => {
    try {
      const response = await axios.post(
        '/api/v1/competition/submit-answer',
        {
          word_id: wordId,
          is_correct: isCorrect,
          time_spent_ms: timeSpent,
          question_type: 'choice'
        },
        {
          headers: {
            Authorization: `Bearer ${localStorage.getItem('token')}`
          }
        }
      );

      // 显示答题反馈
      setFeedbackData(response.data);
      setShowFeedback(true);
    } catch (error) {
      console.error('提交答题失败:', error);
    }
  };

  return (
    <>
      <div>
        {/* 题目内容 */}
        <button onClick={() => handleAnswer(123, true, 2500)}>
          提交答案
        </button>
      </div>

      {/* 答题反馈弹窗 */}
      {showFeedback && feedbackData && (
        <AnswerFeedback
          isCorrect={feedbackData.result.is_correct}
          scoreBreakdown={feedbackData.result}
          rankChange={feedbackData.result}
          comboStatus={feedbackData.combo_status}
          onClose={() => setShowFeedback(false)}
        />
      )}
    </>
  );
}
```

### 3. 监听实时事件

```tsx
import { useEffect } from 'react';
import { competitionWS } from './services/websocket';

function useCompetitionEvents() {
  useEffect(() => {
    // 监听排名更新
    const handleRankUpdate = (message: any) => {
      console.log('排名更新:', message.data);
      // 更新UI
    };

    // 监听被超越
    const handleOvertaken = (message: any) => {
      console.log('被超越了!', message.data);
      // 显示提示
    };

    competitionWS.on('rank_update', handleRankUpdate);
    competitionWS.on('overtaken', handleOvertaken);

    return () => {
      competitionWS.off('rank_update', handleRankUpdate);
      competitionWS.off('overtaken', handleOvertaken);
    };
  }, []);
}
```

## 🎯 积分计算规则

### 公式

```
总得分 = (基础分 + 难度加成 + 速度奖励 + 连击奖励 + 首次奖励) × 连击倍数
```

### 详细说明

| 项目 | 计算方式 | 示例 |
|------|----------|------|
| **基础分** | 固定10分 | 10 |
| **难度加成** | 难度等级 × 5 | 难度3 → 15分 |
| **速度奖励** | 3秒内+10分,每多1秒-2分 | 2秒 → 10分<br>5秒 → 6分 |
| **连击奖励** | 连击数 × 2 (≥2连击) | 5连击 → 10分 |
| **首次奖励** | 首次答对+5分 | 5分 |
| **连击倍数** | 2连击×1.5<br>5连击×2<br>10连击×3 | 5连击时整体×2 |

### 示例计算

**场景**: 难度3单词,答题2.5秒,当前5连击,首次答对

```
基础分 = 10
难度加成 = 3 × 5 = 15
速度奖励 = 10 - (2.5 - 3) × 2 = 9
连击奖励 = 5 × 2 = 10
首次奖励 = 5
连击倍数 = 2.0 (5连击)

总得分 = (10 + 15 + 9 + 10 + 5) × 2.0 = 98分
```

## 🛡️ 防作弊机制

### 1. 答题时间验证

- **最小答题时间**: 500毫秒
- **处理**: 少于500ms的答题会被拒绝

### 2. 答题频率限制

- **限制**: 每分钟最多60次答题
- **处理**: 超过限制的请求会被拒绝

### 3. 重复答题检测

- **限制**: 5秒内不能重复答同一单词
- **处理**: 重复答题会被拒绝

### 4. 异常模式检测

**触发条件**:
- 最近20题正确率 > 98%
- 平均答题时间 < 1秒

**处理**:
- 标记可疑用户
- 后台记录日志
- 可进行人工审核

### 5. 风险评分系统

系统自动计算用户风险分数(0-100):

| 风险项 | 加分 | 说明 |
|--------|------|------|
| 正确率>95% | +20 | 异常高正确率 |
| 连击>50 | +15 | 连击数过高 |
| 平均速度<1秒 | +25 | 答题速度过快 |
| 短时间积分暴涨 | +20 | 10题得分>1000 |
| 答题时间分布异常稳定 | +15 | 标准差<200ms |

**风险等级**:
- 0-29: 低风险 (正常)
- 30-59: 中风险 (警告)
- 60-100: 高风险 (需审核)

## ⚙️ 定时任务配置

### 1. 每日重置任务

每天0点自动执行:
- 保存昨日排行榜快照
- 重置所有用户的`daily_score`
- 清除Redis缓存

### 2. 每周重置任务

每周一0点自动执行:
- 保存上周排行榜快照
- 重置所有用户的`weekly_score`

### 3. 配置方式

在 `backend/app/main.py` 的 lifespan 中启动:

```python
from apscheduler.schedulers.asyncio import AsyncIOScheduler
from app.services.scheduler_tasks import reset_daily_scores, reset_weekly_scores

@asynccontextmanager
async def lifespan(app: FastAPI):
    # 启动调度器
    scheduler = AsyncIOScheduler()
    scheduler.add_job(reset_daily_scores, 'cron', hour=0, minute=0)
    scheduler.add_job(reset_weekly_scores, 'cron', day_of_week='mon', hour=0, minute=0)
    scheduler.start()

    yield

    scheduler.shutdown()
```

## 📈 性能优化建议

### 1. Redis缓存

推荐使用Redis缓存排行榜数据:

```python
import redis
redis_client = redis.Redis(host='localhost', port=6379)

# 缓存排行榜(60秒过期)
async def get_leaderboard_cached(season_id, board_type):
    cache_key = f"leaderboard:{season_id}:{board_type}"
    cached = redis_client.get(cache_key)

    if cached:
        return json.loads(cached)

    # 从数据库查询
    leaderboard = await fetch_from_db(season_id, board_type)
    redis_client.setex(cache_key, 60, json.dumps(leaderboard))

    return leaderboard
```

### 2. 数据库索引

已创建的索引:
- `idx_user_scores_daily`: 每日排名查询
- `idx_user_scores_weekly`: 每周排名查询
- `idx_user_scores_overall`: 总排名查询

### 3. WebSocket连接池

限制单个用户的WebSocket连接数:

```python
# 在ConnectionManager中
if user_id in self.active_connections:
    # 断开旧连接
    await self.disconnect(user_id)
```

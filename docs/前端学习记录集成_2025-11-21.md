## 新增/修改的文件

### 1. 新建API客户端文件

**文件**: `frontend/src/api/learningRecords.ts`

**功能**: 封装所有学习记录相关的API调用

**导出的API函数**:
- `createLearningRecords()` - 批量创建学习记录
- `createStudySession()` - 创建学习会话
- `updateStudySession()` - 更新学习会话
- `getWordMastery()` - 获取单个单词掌握度
- `getAllWordMastery()` - 获取所有单词掌握度
- `getWeakWords()` - 获取薄弱单词
- `getReviewDueWords()` - 获取需要复习的单词

**TypeScript类型定义**:
```typescript
- WordAnswerCreate          // 单个单词答题记录
- LearningRecordBatchCreate // 批量学习记录请求
- StudySessionCreate        // 创建学习会话请求
- StudySessionUpdate        // 更新学习会话请求
- StudySessionResponse      // 学习会话响应
- WordMasteryResponse       // 单词掌握度响应
```

## UI增强功能

### 1. 单词掌握度可视化显示

**位置**: 卡片上方,难度星级旁边

**显示内容**:
- 6个圆点表示0-5级掌握度
- 黄色实心圆点表示已达到的级别
- 白色半透明圆点表示未达到的级别
- 数字显示当前级别 (如 "3/5")

**效果**:
- 学生可以直观看到自己对每个单词的掌握情况
- 激励学生反复练习提升掌握度

```tsx
{currentWordMastery && (
  <motion.div className="inline-flex items-center gap-3 bg-gradient-to-r from-blue-500 to-purple-500 text-white px-6 py-3 rounded-full shadow-lg">
    <span className="text-lg font-medium">掌握度:</span>
    <div className="flex gap-1">
      {[0, 1, 2, 3, 4, 5].map((level) => (
        <div
          key={level}
          className={`w-3 h-3 rounded-full ${
            level < currentWordMastery.mastery_level
              ? 'bg-yellow-300'
              : 'bg-white/30'
          }`}
        />
      ))}
    </div>
    <span className="text-sm font-bold">
      {currentWordMastery.mastery_level}/5
    </span>
  </motion.div>
)}
```

### 2. 薄弱单词智能提醒

**位置**: 页面顶部,断点续学提示之后

**触发条件**:

**显示信息**:

**效果**:

## 数据流程

### 完整的学习数据流:

```
1. 开始学习
   ↓
2. 创建学习会话 (StudySession)
   ↓
3. 显示第一个单词
   ↓
4. 开始计时
   ↓
5. 加载单词掌握度 (如果有历史记录)
   ↓
6. 学生查看单词并翻转卡片
   ↓
7. 学生选择"认识"或"不认识"
   ↓
8. 记录答题时间和结果
   ↓
9. 移动到下一个单词(重复3-8)
   ↓
10. 完成所有单词
    ↓
11. 批量提交所有学习记录
    ↓
12. 后端更新每个单词的掌握度
    ↓
13. 后端计算下次复习时间
    ↓
14. 更新学习会话统计
    ↓
15. 显示完成对话框
```

## 技术亮点

### 1. 性能优化
- **批量提交**: 所有答题记录在学习结束时一次性提交,减少API调用
- **静默失败**: 学习记录API失败不影响主学习流程
- **异步加载**: 掌握度和薄弱单词提醒异步加载,不阻塞主流程

### 2. 用户体验优化
- **实时反馈**: 每个单词都显示当前掌握度
- **智能提醒**: 自动提示需要重点练习的薄弱单词
- **无缝集成**: 学习记录功能不影响原有学习流程
- **渐进式显示**: 使用Framer Motion动画平滑显示掌握度和提醒

### 3. 数据可靠性
- **断点续学**: 结合原有的学习进度保存
- **完整记录**: 记录每个单词的答题时间、正确性、学习模式
- **会话追踪**: 完整记录每次学习会话的统计数据

## 与后端API的对接

### API调用时机:

1. **开始学习时**:
   - `POST /api/v1/student/sessions` - 创建学习会话
   - `GET /api/v1/student/mastery/{word_id}` - 获取单词掌握度
   - `GET /api/v1/student/weak-words?unit_id=X` - 获取薄弱单词

2. **学习过程中**:
   - 答题数据暂存在前端内存中

3. **完成学习时**:
   - `POST /api/v1/student/records` - 批量提交所有学习记录
   - `PUT /api/v1/student/sessions/{id}` - 更新学习会话统计

# 🚀 快速开始指南

## 第一步: 启动后端服务

```bash
cd backend

# 方式1: 使用启动脚本 (推荐)
./start.sh

# 方式2: 手动启动
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

启动后访问:
- API文档: http://localhost:8000/docs
- 健康检查: http://localhost:8000/health

## 第二步: 配置AI (可选)

编辑 `backend/.env` 文件,填入你的API Key:

```env
# OpenAI
OPENAI_API_KEY=sk-xxx
OPENAI_MODEL=gpt-4-turbo-preview

# 或 Claude
ANTHROPIC_API_KEY=sk-ant-xxx
ANTHROPIC_MODEL=claude-3-sonnet-20240229
```

## 第三步: 导入示例数据

```bash
cd backend
python seed_data.py
```

将导入10个示例单词和2个单词本。

## 第四步: 测试API

```bash
cd backend
python test_api.py
```

## 主要API接口速览

### 1. 创建单词 (老师录入)

```bash
curl -X POST "http://localhost:8000/api/v1/words/" \
  -H "Content-Type: application/json" \
  -d '{
    "word": "hello",
    "phonetic": "/həˈləʊ/",
    "difficulty": 1,
    "grade_level": "小学",
    "definitions": [
      {
        "part_of_speech": "int.",
        "meaning": "你好;喂",
        "example_sentence": "Hello, how are you?",
        "example_translation": "你好,你好吗?",
        "is_primary": true
      }
    ],
    "tags": ["日常用语", "问候"]
  }'
```

### 2. 获取单词列表

```bash
# 基础查询
curl "http://localhost:8000/api/v1/words/?limit=10"

# 按年级筛选
curl "http://localhost:8000/api/v1/words/?grade_level=小学"

# 按难度筛选
curl "http://localhost:8000/api/v1/words/?difficulty=2"

# 搜索
curl "http://localhost:8000/api/v1/words/?search=happy"
```

### 3. AI生成例句

```bash
curl -X POST "http://localhost:8000/api/v1/ai/generate-example" \
  -H "Content-Type: application/json" \
  -d '{
    "word": "happy",
    "meaning": "快乐的",
    "difficulty": "primary-school"
  }'
```

### 4. AI生成试卷

```bash
curl -X POST "http://localhost:8000/api/v1/ai/generate-exam" \
  -H "Content-Type: application/json" \
  -d '{
    "word_ids": [1, 2, 3, 4, 5],
    "question_types": ["choice", "fill_blank", "spelling"],
    "total_count": 15
  }'
```

### 5. 批量导入单词

```bash
curl -X POST "http://localhost:8000/api/v1/words/batch-import" \
  -H "Content-Type: application/json" \
  -d '{
    "words": [
      {
        "word": "sun",
        "phonetic": "/sʌn/",
        "difficulty": 1,
        "grade_level": "小学",
        "definitions": [
          {
            "part_of_speech": "n.",
            "meaning": "太阳",
            "is_primary": true
          }
        ],
        "tags": ["自然"]
      }
    ]
  }'
```

## 数据库位置

SQLite数据库文件: `backend/english_helper.db`

可以使用任何SQLite客户端查看:
- DB Browser for SQLite
- SQLite CLI: `sqlite3 english_helper.db`

## 常见问题

### Q: 启动失败,提示端口被占用?
```bash
# 查看占用端口的进程
lsof -i :8000

# 或换个端口启动
uvicorn app.main:app --reload --port 8001
```

### Q: AI功能报错?
检查:
1. `.env` 文件是否配置了API Key
2. API Key是否有效
3. 网络是否能访问AI服务

AI功能是可选的,不影响其他功能使用。

### Q: 如何重置数据库?
```bash
cd backend
rm english_helper.db
# 重新启动服务,会自动创建新数据库
```

### Q: 如何查看日志?
FastAPI会在终端输出所有请求日志。

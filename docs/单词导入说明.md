# 单词批量导入说明

给录入同事看。照着填就不会出错，重点只有一列：**「发音文本」**。

## 一、怎么操作

1. 进「教师端 → 单词本 → 进入某本单词本 → 单元管理」。
2. 点开一个单元，点「下载模板」，得到 `单词导入模板.xlsx`，里面有示例行。
3. 照模板把单词填进去（把示例行删掉，换成自己的），保存。
4. 在同一处点「Excel批量导入」，选这个文件上传。

支持 `.xlsx / .xls`。导入的词会进入**当前这个单元**，单元之间互不影响。

## 二、每一列填什么

模板共 8 列，列名是中文：

| 列名 | 必填 | 填什么 | 例子 |
|---|---|---|---|
| 单词 | ✅ | 单词/短语的正常拼写（学生看到的就是它） | `apple` |
| 音标 | 建议 | 音标（只用于展示） | `/ˈæp.l/` |
| 音节 | 可选 | 音节划分，用 `#` 分隔（决定单词分色） | `ap#ple` |
| 发音文本 | 可选 | **怎么读**（只在读音≠拼写时填，见第三节） | `miz` |
| 词性 | 建议 | 词性 | `n.` |
| 释义 | ✅ | 中文释义 | `苹果` |
| 例句 | 可选 | 例句 | `I like to eat apples.` |
| 例句翻译 | 可选 | 例句翻译 | `我喜欢吃苹果。` |

词性常用值：`n.` 名词、`v.` 动词、`adj.` 形容词、`adv.` 副词、`prep.` 介词、`conj.` 连词、`pron.` 代词、`phr.` 短语。

> 关于「音节」列：用 `#` 把单词按音节断开，如 `beau#ti#ful`、`re#cord`。系统按它给单词的不同音节上不同颜色，帮学生看清结构。**连字符词（self-study）、短语（ice cream）不用断音节，留空即可。** 不填这列也不影响学习，只是少了分色。

## 三、发音怎么填（重点）

系统发音规则很简单：**「发音文本」有值就按它读，留空就按「单词」拼写读。**

### 1. 绝大多数普通词：发音文本留空

`apple`、`book`、`run`…… 直接按拼写就能读对，这一列**空着不用管**。

### 2. 读音和拼写不一样的词：发音文本填"实际怎么念"

用**普通字母**把读音拼出来（不是音标符号 /ɪ/ 这种，机器读不了），让机器照着念。

| 单词 | 发音文本 | 为什么 |
|---|---|---|
| `Ms` | `miz` | 拼写是 Ms，但读 /mɪz/（"米兹"） |
| `Mrs` | `missus` | 读 /ˈmɪsɪz/ |
| `Dr` | `doctor` | 读全称 |
| `kg` | `kilogram` | 读全称 |

> ⚠️ 重要：系统**不会**自己猜缩写怎么读，**一律以你填的「发音文本」为准**。
> - 想读成一个词 → 填那个词的拼法（Ms → `miz`，不要填 `Mzi` 这种不符合英文拼读的）。
> - 缩写不填发音文本，就会被一个字母一个字母念出来。

### 3. 要一个字母一个字母念的缩写

如果这个缩写就是要念字母（如 `MS` 念 "M-S"、`PE` 念 "P-E"、`ID` 念 "I-D"），**发音文本填字母、中间加空格**：

| 单词 | 发音文本 | 读出来 |
|---|---|---|
| `MS` | `M S` | "M—S"（逐字母） |
| `PE` | `P E` | "P—E" |
| `ID` | `I D` | "I—D" |

一句话区分：**读成一个词 → 填那个词的发音（Ms→miz）；要念字母 → 字母间加空格（M S）。**

### 4. 占位符 sb / sth

短语里的 `sb` / `sth` 系统会自动念成 somebody / something，**照常写就行**。如 `help sb with sth` 直接填，不用展开。

## 四、一个词有两种读音怎么办（同形异音）

### 情况 A：读音不同 + 意思也不同 → 拆成两行（最常见）

有些词拼写一样，但词性不同、读音和意思都不同：

- `record`：名词 /ˈrekɔːd/（记录、唱片）；动词 /rɪˈkɔːd/（录制）
- `read`：现在式 /riːd/；过去式 /red/
- 类似还有 `present`、`object`、`lead`、`live`、`tear`…

**做法：拆成两行，「单词」列都写同一个拼写，其余各填各的：**

| 单词 | 音标 | 音节 | 发音文本 | 词性 | 释义 |
|---|---|---|---|---|---|
| `record` | `/ˈrek.ɔːd/` | `re#cord` | `rekord` | `n.` | 记录;唱片 |
| `record` | `/rɪˈkɔːd/` | `re#cord` | `rikord` | `v.` | 录制 |

这样名词和动词会作为**两个独立词条**分别出现、各自读对音。（系统已支持，导入后两条都在，不会互相覆盖。）

### 情况 B：读音略有不同，但意思一样 → 只填一行

像 `either`（/ˈaɪðə/ 或 /ˈiːðə/，都是"任一"）、`often`（t 可发可不发）这种——意思相同，**只填一行**，选教材标准的那个读音填进「发音文本」即可，不用拆行。

## 五、自检清单（上传前过一遍）

- [ ] 「单词」和「释义」都填了（这两列必填）
- [ ] 「单词」列填的是正常拼写（学生要拼的就是它，别把读音写进这列）
- [ ] 缩写/特殊读法的词，「发音文本」填了实际读音（读词→填词的拼法；读字母→字母间加空格）
- [ ] 一词多音（意思也不同的）拆成了多行，单词列写同一拼写
- [ ] 普通词的「发音文本」**留空**（不要乱填，否则反而读错）
- [ ] 音节列如果填了，用 `#` 分隔（`ap#ple`），没填也没关系

# 单词批量导入说明

给录入同事看。照着填就不会出错，重点只有一列：**`tts_text`（发音文本）**。

## 一、怎么操作

1. 进「教师端 → 单词管理 → 批量导入」。
2. 点「下载模板」，得到 `单词导入模板.xlsx`，里面有示例行。
3. 照模板把单词填进去，保存。
4. 上传文件 → 系统解析 → 确认无误后点「导入」。

支持 `.xlsx / .xls / .csv`，单次建议不超过 1000 个词。同一个词已存在会自动跳过。

## 二、每一列填什么

| 列名 | 必填 | 填什么 | 例子 |
|---|---|---|---|
| `word` | ✅ | 单词/短语的正常拼写（学生看到的） | `apple` |
| `phonetic` | ✅ | 音标（展示用） | `/ˈæpl/` |
| `part_of_speech` | ✅ | 词性 | `n.` |
| `meaning` | ✅ | 中文释义 | `苹果` |
| `difficulty` | ✅ | 难度 1–5（1 最易） | `1` |
| `grade_level` | ✅ | 年级：小学 / 初中 / 高中 | `小学` |
| `syllables` | 可选 | 音节划分，用 `-` 分隔 | `beau-ti-ful` |
| `tts_text` | 可选 | **怎么读**（只在读音≠拼写时填，见下） | `miz` |
| `example_sentence` | 可选 | 例句 | `I eat an apple.` |
| `example_translation` | 可选 | 例句翻译 | `我吃一个苹果。` |
| `tags` | 可选 | 标签，逗号分隔 | `水果,常用` |

词性可选值：`n.` 名词、`v.` 动词、`adj.` 形容词、`adv.` 副词、`prep.` 介词、`conj.` 连词、`pron.` 代词。

## 三、发音怎么填（重点）

系统发音的规则很简单：**`tts_text` 有值就按它读，留空就按 `word` 拼写读。**

### 1. 绝大多数普通词：`tts_text` 留空

`apple`、`book`、`run`……直接按拼写就能读对，这一列**空着不用管**。

### 2. 读音和拼写不一样的词：`tts_text` 填"实际怎么念"

用**普通字母**把读音拼出来（不是音标符号），让机器照着念。

| word | tts_text | 为什么 |
|---|---|---|
| `Ms` | `miz` | 拼写是 Ms，但读 /mɪz/ |
| `Mrs` | `missus` | 读 /ˈmɪsɪz/ |
| `Dr` | `doctor` | 读全称 |
| `kg` | `kilogram` | 读全称 |

> ⚠️ 重要：系统**不会**自己猜缩写怎么读。因为同一个缩写可能有两种读法（见下），机器猜会出错，所以**一律以你填的 `tts_text` 为准**。缩写词不填 `tts_text`，就会被一个字母一个字母念出来。

### 3. 同一个缩写，要一个字母一个字母念

如果这个缩写就是要念字母（如 `MS` 念 "M-S"、`PE` 念 "P-E"、`ID` 念 "I-D"），**`tts_text` 填字母、中间加空格**：

| word | tts_text | 读出来 |
|---|---|---|
| `MS` | `M S` | "M—S"（字母） |
| `PE` | `P E` | "P—E" |
| `ID` | `I D` | "I—D" |

一句话区分：**读成一个词 → 填那个词的发音（Ms→miz）；要念字母 → 字母间加空格（M S）。**

### 4. 占位符 sb / sth

短语里的 `sb` / `sth` 系统会自动念成 somebody / something，**照常写就行**，不用展开。如 `help sb with sth` 直接填。

## 四、一个词有两种读音怎么办（同形异音）

有些词拼写一样、词性不同、读音也不同：

- `record`：名词 /ˈrekɔːd/（记录、唱片）；动词 /rɪˈkɔːd/（录制）
- `read`：现在式 /riːd/；过去式 /red/
- 类似还有 `present`、`object`、`lead`…

**做法：拆成两行录入**，`word` 都写同一个拼写，其余各填各的：

| word | tts_text | phonetic | part_of_speech | meaning |
|---|---|---|---|---|
| `record` | `rekord` | `/ˈrekɔːd/` | `n.` | 记录;唱片 |
| `record` | `rikord` | `/rɪˈkɔːd/` | `v.` | 记录;录制 |

这样名词和动词会作为两个词条分别出现、各自读对音。

## 五、自检清单（上传前过一遍）

- [ ] 6 个必填列都填了（word / phonetic / part_of_speech / meaning / difficulty / grade_level）
- [ ] 难度是 1–5，年级是 小学/初中/高中
- [ ] 缩写、特殊读法的词，`tts_text` 填了实际读音（读词 → 词；读字母 → 字母间加空格）
- [ ] 一词多音的，拆成了多行
- [ ] 普通词的 `tts_text` 留空（不要乱填）

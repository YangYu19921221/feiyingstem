### **阶段2: 搭建前端项目**

**目标**: 创建React项目,配置基础环境

**技术栈**:
```
React 18 + TypeScript
Vite (构建工具)
Tailwind CSS (样式)
Framer Motion (动画)
React Query (数据管理)
Zustand (状态管理)
Axios (HTTP请求)
```

**步骤**:
```bash
# 创建项目
npm create vite@latest frontend -- --template react-ts

cd frontend

# 安装依赖
npm install
npm install -D tailwindcss postcss autoprefixer
npm install framer-motion
npm install @tanstack/react-query
npm install zustand
npm install axios

# 初始化Tailwind
npx tailwindcss init -p
```

**配置文件**:
1. `tailwind.config.js` - 配置主题色(活力橙黄)
2. `src/api/client.ts` - 配置axios baseURL
3. `src/stores/` - 创建状态管理
4. `src/components/` - 创建组件库

**耗时**: 30-60分钟

## 💡 开发建议

### 前端开发顺序:
1. **先硬编码数据** - 快速实现UI
2. **再接入API** - 确保数据流畅
3. **最后优化** - 动画、性能、细节

### 后端开发顺序:
1. **先用Swagger测试** - 确保API正确
2. **再前端调用** - 避免前后端同时调试
3. **遇到问题查日志** - FastAPI会打印详细错误

### 遇到问题:
- 前端问题: React DevTools + Console
- 后端问题: 查看终端日志
- 数据库问题: DB Browser for SQLite

# 竞赛题目系统设计方案

## 一、系统概述

### 1.1 目标
- 支持教师手动上传题目
- 支持AI自动生成题目
- 支持多种题型:选择题、填空题、拼写题、阅读理解
- 题目可用于竞赛模式和日常练习
- 支持题目难度分级和标签管理

### 1.2 核心功能
1. **教师端**
   - 题目管理(CRUD)
   - 批量导入题目
   - AI一键生成题目
   - 题目预览和测试
   - 题库统计分析

2. **学生端**
   - 竞赛模式答题
   - 题目展示和交互
   - 实时反馈
   - 成绩统计

## 二、数据库设计

### 2.1 竞赛题库表 (competition_questions)

```sql
CREATE TABLE competition_questions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,

    -- 基本信息
    question_type VARCHAR(20) NOT NULL, -- choice, fill_blank, spelling, reading
    title TEXT,  -- 题目标题(阅读理解用)
    content TEXT NOT NULL,  -- 题目内容/题干
    passage TEXT,  -- 阅读理解文章

    -- 答案相关
    correct_answer TEXT NOT NULL,  -- 正确答案(JSON格式)
    answer_explanation TEXT,  -- 答案解析

    -- 元数据
    difficulty VARCHAR(20) DEFAULT 'medium',  -- easy, medium, hard
    word_id INTEGER,  -- 关联单词ID(如果基于单词)
    unit_id INTEGER,  -- 关联单元ID
    tags VARCHAR(255),  -- 标签(逗号分隔)

    -- 创建信息
    created_by INTEGER NOT NULL,  -- 创建者ID(教师)
    source VARCHAR(20) DEFAULT 'manual',  -- manual(手动), ai(AI生成)
    is_active BOOLEAN DEFAULT TRUE,  -- 是否启用

    -- 统计
    use_count INTEGER DEFAULT 0,  -- 使用次数
    correct_count INTEGER DEFAULT 0,  -- 答对次数
    total_attempts INTEGER DEFAULT 0,  -- 总答题次数
    avg_time INTEGER DEFAULT 0,  -- 平均答题时间(秒)

    -- 时间戳
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,

    FOREIGN KEY (word_id) REFERENCES words(id) ON DELETE SET NULL,
    FOREIGN KEY (unit_id) REFERENCES units(id) ON DELETE SET NULL,
    FOREIGN KEY (created_by) REFERENCES users(id) ON DELETE CASCADE,

    CHECK(question_type IN ('choice', 'fill_blank', 'spelling', 'reading')),
    CHECK(difficulty IN ('easy', 'medium', 'hard')),
    CHECK(source IN ('manual', 'ai'))
);

CREATE INDEX idx_competition_questions_type ON competition_questions(question_type);
CREATE INDEX idx_competition_questions_difficulty ON competition_questions(difficulty);
CREATE INDEX idx_competition_questions_word ON competition_questions(word_id);
CREATE INDEX idx_competition_questions_unit ON competition_questions(unit_id);
CREATE INDEX idx_competition_questions_active ON competition_questions(is_active);
```

### 2.2 题目选项表 (competition_question_options)

```sql
CREATE TABLE competition_question_options (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    question_id INTEGER NOT NULL,
    option_key VARCHAR(10) NOT NULL,  -- A, B, C, D
    option_text TEXT NOT NULL,
    is_correct BOOLEAN DEFAULT FALSE,
    display_order INTEGER DEFAULT 0,

    FOREIGN KEY (question_id) REFERENCES competition_questions(id) ON DELETE CASCADE
);

CREATE INDEX idx_question_options_question ON competition_question_options(question_id);
```

### 2.3 题目集合表 (competition_question_sets)

```sql
CREATE TABLE competition_question_sets (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    created_by INTEGER NOT NULL,
    total_questions INTEGER DEFAULT 0,
    is_public BOOLEAN DEFAULT FALSE,  -- 是否公开
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP,

    FOREIGN KEY (created_by) REFERENCES users(id) ON DELETE CASCADE
);
```

### 2.4 题目集合关联表 (question_set_items)

```sql
CREATE TABLE question_set_items (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    set_id INTEGER NOT NULL,
    question_id INTEGER NOT NULL,
    display_order INTEGER DEFAULT 0,

    FOREIGN KEY (set_id) REFERENCES competition_question_sets(id) ON DELETE CASCADE,
    FOREIGN KEY (question_id) REFERENCES competition_questions(id) ON DELETE CASCADE,

    UNIQUE(set_id, question_id)
);
```

### 2.5 更新answer_records表

```sql
ALTER TABLE answer_records ADD COLUMN question_id INTEGER;
ALTER TABLE answer_records ADD FOREIGN KEY (question_id) REFERENCES competition_questions(id);
```

## 三、题目格式规范

### 3.1 选择题 (choice)

```json
{
  "question_type": "choice",
  "content": "The word 'happy' means ___",
  "word_id": 123,
  "difficulty": "easy",
  "correct_answer": "A",
  "options": [
    {"key": "A", "text": "快乐的", "is_correct": true},
    {"key": "B", "text": "悲伤的", "is_correct": false},
    {"key": "C", "text": "生气的", "is_correct": false},
    {"key": "D", "text": "害怕的", "is_correct": false}
  ],
  "answer_explanation": "'happy' 表示快乐的、幸福的。"
}
```

### 3.2 填空题 (fill_blank)

```json
{
  "question_type": "fill_blank",
  "content": "I am very ___ to see you. (我很高兴见到你)",
  "word_id": 123,
  "difficulty": "medium",
  "correct_answer": ["happy", "glad", "pleased"],
  "answer_explanation": "根据语境,这里应填入表示'高兴'的形容词。"
}
```

### 3.3 拼写题 (spelling)

```json
{
  "question_type": "spelling",
  "content": "请拼写下列单词: /ˈhæpi/ (快乐的)",
  "word_id": 123,
  "difficulty": "easy",
  "correct_answer": "happy",
  "answer_explanation": "happy [ˈhæpi] adj. 快乐的"
}
```

### 3.4 阅读理解 (reading)

```json
{
  "question_type": "reading",
  "title": "My Happy Day",
  "passage": "Today is a happy day. I went to the park with my friends...",
  "content": "What did the author do today?",
  "difficulty": "medium",
  "correct_answer": "A",
  "options": [
    {"key": "A", "text": "Went to the park", "is_correct": true},
    {"key": "B", "text": "Stayed at home", "is_correct": false},
    {"key": "C", "text": "Went to school", "is_correct": false},
    {"key": "D", "text": "Visited grandparents", "is_correct": false}
  ],
  "answer_explanation": "文章第一句提到'I went to the park with my friends'。"
}
```

## 四、API设计

### 4.1 教师端API

#### 创建题目
```
POST /api/v1/teacher/competition-questions
```

#### 获取题目列表
```
GET /api/v1/teacher/competition-questions?type=choice&difficulty=easy&page=1&limit=20
```

#### 更新题目
```
PUT /api/v1/teacher/competition-questions/{question_id}
```

#### 删除题目
```
DELETE /api/v1/teacher/competition-questions/{question_id}
```

#### AI生成题目
```
POST /api/v1/teacher/competition-questions/ai-generate
Body: {
  "word_ids": [1, 2, 3],
  "question_types": ["choice", "fill_blank"],
  "difficulty": "medium",
  "count": 10
}
```

#### 批量导入题目
```
POST /api/v1/teacher/competition-questions/batch-import
Body: FormData with Excel file
```

### 4.2 学生端API

#### 获取竞赛题目
```
GET /api/v1/competition/questions/random?count=10&difficulty=medium&type=choice
```

#### 提交答案
```
POST /api/v1/competition/submit-answer
Body: {
  "question_id": 123,
  "user_answer": "A",
  "time_spent": 5000
}
```

## 六、AI出题功能

### 6.1 支持的题型生成

#### 选择题生成
- 根据单词生成词义选择题
- 根据例句生成语法选择题

- 给定音标拼写单词
- 给定中文释义拼写单词

#### 阅读理解生成
- 生成短文并配套问题
- 基于单元词汇生成阅读材料

### 6.2 AI Prompt设计

```python
def generate_choice_question(word, difficulty):
    prompt = f"""
作为英语教学助手,请为以下单词生成一道选择题:

单词: {word.word}
释义: {word.meaning}
难度: {difficulty}

要求:
1. 题干简洁明了
2. 4个选项,只有1个正确答案
3. 干扰项要有一定迷惑性
4. 符合{difficulty}难度
5. 适合中小学生

请以JSON格式返回:
{{
  "content": "题干",
  "options": [
    {{"key": "A", "text": "选项A", "is_correct": true}},
    {{"key": "B", "text": "选项B", "is_correct": false}},
    {{"key": "C", "text": "选项C", "is_correct": false}},
    {{"key": "D", "text": "选项D", "is_correct": false}}
  ],
  "answer_explanation": "答案解析"
}}
"""
    return prompt
```

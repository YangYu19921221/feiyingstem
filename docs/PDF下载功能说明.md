# PDF下载功能说明

## 功能概述

系统现在支持将试卷导出为专业的PDF格式,适合打印和分发。

## ✨ 新增功能

### 1. PDF格式下载
- 📄 标准A4纸张大小
- 📋 完整的试卷头部信息
- 📑 自动分页(试卷+答案)
- 🎨 专业的排版格式
- 💾 一键下载到本地

### 2. 多格式支持

系统现在提供3种下载选项:

| 格式 | 用途 | 特点 |
|------|------|------|
| **PDF** | 打印、分发 | 专业格式,包含答案页 |
| **TXT** | 文本编辑 | 纯文本,易于修改 |
| **打印** | 直接打印 | 浏览器打印功能 |

### 3. PDF内容结构

生成的PDF包含:

#### 第一页: 试卷正文

```
------------------------------------------

一、选择题 (共8题,每题5分,共40分)
1. ...
2. ...

二、填空题 (共6题,每题5分,共30分)
9. ...
10. ...

三、拼写题 (共4题,每题5分,共20分)
15. ...

四、阅读理解 (共2题,每题5分,共10分)
```

#### 第二页: 参考答案

## 🎯 使用方法

### 步骤1: 生成试卷
1. 教师端 → 数据分析 → 学生详情
2. 点击"一键生成试卷"
3. 等待AI生成完成(带进度提示)

### 步骤2: 预览试卷
- 查看试卷内容
- 点击"显示/隐藏答案"切换答案显示
- 确认试卷无误

### 步骤3: 下载PDF
1. 点击顶部工具栏的"下载PDF"按钮
2. 等待PDF生成(3-5秒)
3. PDF自动下载到本地

## 🔧 技术实现

### 使用的库
```json
{
  "html2pdf.js": "^0.10.1"
}
```

### 核心配置
```typescript
const options = {
  margin: [10, 10, 10, 10],
  filename: `${exam.title}.pdf`,
  image: { type: 'jpeg', quality: 0.95 },
  html2canvas: {
    scale: 2,
    useCORS: true,
    letterRendering: true,
  },
  jsPDF: {
    unit: 'mm',
    format: 'a4',
    orientation: 'portrait',
    compress: true
  },
  pagebreak: {
    before: '.page-break-before'
  }
};
```

### 生成流程
1. **准备内容**: 临时显示答案页
2. **渲染HTML**: html2canvas转图片
3. **生成PDF**: jsPDF创建文档
4. **下载文件**: 自动触发下载
5. **恢复状态**: 隐藏答案页

## 🐛 常见问题

### Q: PDF生成失败怎么办?
A:
1. 检查浏览器控制台是否有错误
2. 确保网络连接正常
3. 尝试刷新页面重新生成
4. 使用"打印"功能作为替代

### Q: PDF文件太大?
A:
- 当前配置已优化压缩
- 典型试卷约200-500KB
- 如需更小,可降低图片质量

### Q: 答案页没有?
A:
- PDF自动包含答案页
- 答案页在第二页
- 检查PDF总页数

### Q: 格式不对?
A:
- 确保使用最新版浏览器
- 清除浏览器缓存
- 尝试不同浏览器

### Q: 可以编辑PDF吗?
A:
- 生成的PDF是图片格式
- 不可直接编辑
- 建议下载TXT格式后编辑
